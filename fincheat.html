# 항목 공식 의미 / 활용
1 현재가치Present Value, PV PV = FV / (1 + r)ⁿ 미래 현금흐름을 요구수익률로 할인한 현재 시점의 가치.
2 미래가치Future Value, FV FV = PV × (1 + r)ⁿ 현재 투자금이 복리로 불어나 미래에 얼마가 되는지를 보여줌.
3 순현재가치Net Present Value, NPV NPV = Σ CFₜ / (1 + r)ᵗ 할인된 현금유입과 유출을 비교해 투자가 창출하는 가치를 측정.
4 가중평균자본비용WACC WACC = (E/V)Re + (D/V)Rd(1−Tc) 자기자본과 타인자본을 가중평균한 자본비용으로, 밸류에이션 할인율로 사용.
5 자본자산가격결정모형CAPM Re = Rf + β(Rm − Rf) 베타(시장 위험)를 기반으로 투자자가 요구하는 기대수익률을 추정.
6 주당순이익EPS EPS = 순이익 / 유통주식수 보통주 1주당 회사가 벌어들이는 이익 규모.
7 주가수익비율P/E Ratio P/E = 주가 / EPS 투자자가 주당 순이익 1원당 얼마를 지불할 의향이 있는지를 나타냄.
8 부채비율Debt-to-Equity D/E = 총부채 / 자기자본 자기자본 대비 부채 규모로 재무 레버리지를 측정.
9 자기자본이익률ROE ROE = 순이익 / 자기자본 주주 투자금을 활용해 얼마나 효율적으로 이익을 내는지 측정.
10 총자산수익률ROA ROA = 순이익 / 총자산 총자산을 활용해 얼마나 효과적으로 이익을 창출하는지 나타냄.
11 투자수익률ROI ROI = (투자수익 − 비용) / 비용 투입한 비용 대비 투자에서 얻은 수익률을 측정.
12 유동비율Current Ratio 유동비율 = 유동자산 / 유동부채 유동자산으로 단기 채무를 상환할 수 있는 능력을 측정.
13 당좌비율Quick Ratio 당좌비율 = (유동자산 − 재고) / 유동부채 재고를 제외한 더 엄격한 단기 유동성 지표.
14 에비타EBITDA EBITDA = 순이익 + 이자 + 세금 + 감가상각 회계·금융 효과를 제외한 핵심 영업이익을 보여줌.
15 투자회수기간Payback Period 회수기간 = 초기투자금 / 연간현금유입 초기 투자금을 현금흐름으로 회수하는 데 걸리는 시간.
16 배당수익률Dividend Yield 배당수익률 = 연간 주당배당금 / 주가 주가 대비 연간 배당소득의 비율.
17 기업가치Enterprise Value, EV EV = 시가총액 + 총부채 + 우선주 − 현금성자산 부채와 우선주를 포함하고 현금을 제외한 기업 전체 가치.
18 이자보상비율Interest Coverage 이자보상비율 = EBIT / 이자비용 영업이익으로 이자비용을 얼마나 여유 있게 감당하는지 측정.
19 샤프비율Sharpe Ratio Sharpe = (Rp − Rf) / σp 총위험(변동성) 대비 초과수익을 비교한 위험조정수익률.
20 베타Beta, β β = Cov(Ri, Rm) / Var(Rm) 시장 전체 움직임에 대한 개별 종목의 체계적 위험(민감도).
